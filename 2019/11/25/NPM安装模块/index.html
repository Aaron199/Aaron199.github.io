    1.2 NPM安装模块

    NPM安装模块分为本地安装安装和全局安装

    • 本地安装: 将JS库文件安装在当前执行命令的目录下
    • 全局安装: 将JS库文件安装在全局目录下

    NPM按照生产环境安装模块分为生产环境安装和开发环境安装

    之所以有这样的区分是因为有些依赖包并不需要再生产环境安装
